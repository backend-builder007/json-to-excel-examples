# JSON to Excel Examples

This repository contains sample JSON files and example structures that can be converted into Excel spreadsheets.

Tool:
https://jsontoexcel.pro

## What the tool does

- Converts JSON to Excel (.xlsx)
- Handles nested JSON structures
- Flattens objects into spreadsheet columns
- Supports multi-sheet Excel output

## Example JSON

See the included `sample.json` file for a basic nested JSON example.
